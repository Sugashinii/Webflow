# Noctis — AI Analytics Platform 🌑

A dark minimal SaaS landing page for **Noctis**, a fictional AI-powered data analytics platform. Built to demonstrate modern frontend design principles including responsive layouts, CSS animations, and clean component structure.

🔗 **Live Demo:** [https://zesty-croquembouche-d63636.netlify.app](https://zesty-croquembouche-d63636.netlify.app)

---

## 📸 Preview

> Dark minimal SaaS landing page with lime green accents, animated hero, feature grid, dashboard mockup, and pricing cards.

---

## ✨ Features

- **Custom animated cursor** with trailing ring effect
- **Scroll reveal animations** — elements fade up as you scroll
- **Animated marquee strip** — scrolling feature tags
- **Interactive dashboard mockup** with animated bar chart
- **Fully responsive** — mobile, tablet, desktop
- **CSS variables** for consistent theming
- **Noise texture overlay** for depth and atmosphere
- **Glassmorphism navbar** with backdrop blur

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables, Animations)
- Vanilla JavaScript
- Google Fonts (Syne + DM Mono)

## 🎨 Design System

| Token | Value | Usage |
|-------|-------|-------|
| `--bg` | `#080808` | Page background |
| `--surface` | `#0f0f0f` | Card backgrounds |
| `--accent` | `#c8f542` | Lime green — buttons, highlights |
| `--text` | `#e8e8e0` | Primary text |
| `--muted` | `#5a5a52` | Secondary text |

**Fonts:**
- Display: `Syne` (headings)
- Mono: `DM Mono` (labels, code, stats)

---

## 📄 Sections

1. **Navbar** — Fixed, blur background, animated on load
2. **Hero** — Full viewport, bold typography, stat strip
3. **Marquee** — Scrolling feature tags
4. **Features Grid** — 6 cards in 3-column CSS Grid
5. **Dashboard Preview** — Simulated app UI with KPIs and bar chart
6. **Pricing** — 3-tier cards (Free, Pro $79, Enterprise)
7. **CTA** — Final call to action with glow effect
8. **Footer** — Minimal links
